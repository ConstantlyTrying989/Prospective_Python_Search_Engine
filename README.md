# Python Text Search Engine

## Overview
This project implements a text-based search engine from scratch in Python.
It focuses on document indexing, ranking, and evaluation using classical
information retrieval techniques.

## Dataset
Public-domain text corpus (Project Gutenberg subset).

## Methods
- Text preprocessing
- TF-IDF vectorization
- Cosine similarity ranking

## Current Status
Project setup and planning phase.

## Planned Extensions
- Evaluation metrics
- Performance improvements
- Domain-specific search experiments
